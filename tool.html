🧪 NMA GRADE 評価サポートツール（R Shiny版）
NMAにおけるエビデンスの確実性の評価サポートツールとして、統計ソフトのRを利用したShinyアプリです。 データ読込・NMA実行・GRADE評価（A1〜D）・SoFテーブル出力まで、一連のフローをGUI操作で実施できます。

📥 ダウンロード

単一ファイル版（app.R）をダウンロードして、RStudio または Rコンソールから実行してください。
📂 GitHub でファイルを確認する ↗

🚀 起動方法

RStudio から起動する場合

  1. app.R を RStudio で開く
  2. 右上の「Run App」ボタンをクリック

R コンソールから起動する場合

# 作業ディレクトリを app.R のある場所に変更（自分のPC環境に合わせて変更） setwd("F:/マイドライブ/2016年works/AI/作成アプリ/NMA確実性") shiny::runApp("app.R")

🔄 使用フロー（ステップ順）

0. プロジェクト設定　 効果指標（OR/RR等）・アウトカム・閾値を設定
↓
1. データ読込　 CSVアップロード or サンプルデータ（smoking 等）を読込
↓
2. NMA実行　 参照介入・推定法を選択して「NMA実行」ボタン
─── 解析結果確認 ───
ネットワーク図 → Forest Plot → netsplit → リーグテーブル → ランキング
─── GRADE評価（順番に実施） ───
A1  A2  → B1  B2  B3  → C1  C2  C3  → D
─── 結果出力 ───
SoFテーブル → 色分け図 → 保存 / 読込

📋 各ステップの概要

ステップ内容
A1 直接比較の4ドメイン評価（RoB・非一貫性・非直接性・出版バイアス）
A2 直接推定値の不精確さ（非整合性 Serious の場合のみ）
B1 ループ選択（間接エビデンスの出発点）
B2 非推移性の評価
B3 間接推定値の不精確さ（非整合性 Serious の場合のみ）
C1 NMA出発点の選択（直接 vs 間接）
C2 非整合性（netsplit p値）
C3 NMA推定値の不精確さ
D 最良推定値の選択

⚡ 初回テスト用クイックスタート

サンプルデータ（smoking）を使って動作確認をする手順です。

  1. 「1. データ読込」→ サンプルデータ「smoking」を選択 → 読込
  2. 「2. NMA実行」→ 参照介入を選択 → 「NMA実行」ボタン
  3. 「A1. 直接4ドメイン」→ 一括「Not serious」→「一括適用」→「保存」
  4. 「C2. 非整合性」→「p≥0.10 一括 NS」
  5. 「C3. NMA不精確さ」→「自動判定実行」
  6. 「D. 最良推定値」→「自動判定」
  7. 「SoFテーブル」→ 結果確認

📦 使用パッケージ

📊 統計解析・NMA関連

  • netmeta — ネットワークメタアナリシスの頻度論的実行。
    netmeta(), netsplit(), netrank(), netleague() 等
  • meta — ペアワイズメタアナリシス。netmetaの依存パッケージ。
    metagen(), forest(), funnel() 等
  • NMA — サンプルデータセット提供。
    smoking, heartfailure 等

🖥️ Shinyアプリ関連

  • shiny — インタラクティブウェブアプリフレームワーク
  • shinydashboard — ダッシュボードレイアウト
  • shinyWidgets — 追加UIウィジェット
  • shinyjs — JavaScript操作

📈 データ操作・可視化

  • dplyr — データ操作（case_when() 等）
  • tidyr — データ整形
  • ggplot2 — 色分け図（Phillips et al.準拠）の描画
  • DT — インタラクティブなデータテーブル表示

💾 入出力

  • openxlsx — Excelファイル（.xlsx）の出力
  • jsonlite — JSON形式での保存・読込
