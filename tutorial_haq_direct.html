📋 臨床シナリオ

あなたはDMARD-naïve（MTX未使用）関節リウマチ患者に対する生物学的製剤の有効性を評価する living NMA のGRADE担当者です。

アウトカム: HAQ-DI（Health Assessment Questionnaire-Disability Index）— 身体機能の連続アウトカム（値が低いほど良好）

比較: 特定の生物学的製剤 vs メトトレキサート（MTX）単剤

フォレストプロットと各研究の情報を確認し、各GRADEドメインを評価してください。

⚠️ ステップ2: バイアスリスク（Study Limitations）自動判定あり
研究RoB評価重み主な懸念事項

🤖 自動判定アルゴリズムの計算過程:

Some Concerns または High RoB の累積重み
= Trial_A（38%）+ Trial_C（25%）= 63%（50%超）

High RoB のみの累積重み = 0%（50%未満）

→ ルール: Some Concernsの累積重みが50%超、かつHigh RoBが50%未満
→ 自動判定: 「Some Concerns」フラグ

❓ 質問: 自動判定は「Some Concerns」としています。フォレストプロットを見ると3研究の点推定値は -0.22〜-0.28 と非常に近接しています。Low RoB の Trial_B（重み37%）も同様の結果です。あなたの最終判断は？

🔄 ステップ3: 不一致性（Inconsistency）自動判定あり
🤖 自動判定アルゴリズム:

I² = 12%（閾値 30% 未満）
→ ルール: I² < 30% → 自動判定: 「Not Serious（深刻でない）」
⚡ 実際の不一致例（Appendix 4 HAQ の行より）:
論文中では自動判定が 「Serious」 と出力されていました（計算条件の違いによる）。
評価者Glenはフォレストプロットを確認し 「Not Serious」 に変更。
評価者Samも最終的に同意。
→ 最終判定: Not Serious（確実性が Moderate → High に向上）

フォレストプロットと統計値から、不一致性を評価してください。

参考: I²=12%、p=0.32（有意な異質性なし）、点推定値 -0.22〜-0.28（非常に近接）

🤖 自動判定アルゴリズム（Appendix 2a より）:

判定ルール:
・MTX ≤ 10 mg/week → Some Concerns（Trial_A 該当）

Some Concerns の累積重み = 38%（30%〜50% の範囲）
High indirectness の累積重み = 0%

→ ルール: Some Concerns 重みが 30〜50% の場合はフラグのみ、自動ダウングレードなし
→ 自動判定: 「Not Serious」

学習のポイント:

  • 自動判定はあくまで出発点。フォレストプロットの視覚的確認が重要
  • I²=12%、p=0.32、点推定値の近似 → 不一致性「Not Serious」が明確
  • 自動の「Serious」誤判定を修正したことで Moderate → High に向上
  • Some Concerns 研究が63%の重みでも、点推定値が一致していれば影響は限定的
